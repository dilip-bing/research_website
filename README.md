# Nova Research Labs - Website Base

A modern, research-focused website starter inspired by top-tier industrial research labs.

## Included

- Premium hero section with metrics panel
- Research focus cards
- Publications list layout
- Team section
- Collaboration CTA and polished footer
- Responsive design for desktop, tablet, and mobile
- Scroll reveal animations

## Project Structure

- `index.html` - page structure
- `styles.css` - visual system and responsive layout
- `script.js` - reveal-on-scroll interaction

## Run Locally

Open `index.html` directly in a browser, or run a simple local server:

```bash
python3 -m http.server 8080
```

Then visit:

```text
http://localhost:8080
```

## Customization Ideas

- Replace placeholder partner and publication names with your real data
- Add case-study pages for each research track
- Connect contact CTA to a real form backend
- Add an insights/blog section for announcements
