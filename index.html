NeurIPS 2026 Under Review · Preprint

Neuro-Symbolic Planning via Adversarial Repair and Compilation

Dilip Kumar Thirukonda Chandrasekaran

A closed-loop system that reframes LLM hallucination as a code repair problem. Using compiler-guided adversarial feedback loops, the system iteratively regenerates valid PDDL plans from natural language goals — consistently outperforming zero-shot LLM baselines on IPC benchmark datasets across multiple planning domains.

Research Areas

Intelligent automation and symbolic AI systems

LLM Automation

LLM-driven web & mobile automation

Replacing brittle CSS/XPath selectors with LLM reasoning engines for dynamic, adaptive test automation. Building modular frameworks that leverage frontier models for intelligent UI interaction without fragile dependencies.

Neuro-Symbolic

Hybrid planning with adversarial LLM repair

Combining symbolic planning (PDDL) with LLMs to tackle hallucination through adversarial repair loops. Treating LLM generation errors as compiler faults, not AI failures — enabling iterative convergence to valid plans.

Observability

LLM tracing and evaluation at scale

Deep instrumentation of LLM application stacks with Langfuse and OpenTelemetry. Cost attribution per prompt template, latency bottleneck detection, and automated LLM-as-a-judge evaluation pipelines for production AI systems.

Mobile Testing

E2E automation frameworks for iOS & Android

Architecting native automation frameworks using Espresso, XCTest, and Kaspresso. 10K+ test cases across 5+ shipped products. Full CI/CD integration via GitHub Actions, Docker, and custom device farm infrastructure.

Memory & RAG

Personalized AI memory and knowledge retrieval

Building retrieval-augmented generation systems that learn and grow with users. Exploring memory scope hierarchies, AI-driven memory suggestions, and document intelligence for long-horizon conversational context.

Autonomous Agents

Multi-step agentic task architectures

Designing LLM agent pipelines with tool-calling, routing logic, and self-reflection. Exploring ReAct patterns, Tree of Thoughts, and meta-prompting for complex, multi-step reasoning across real-world task domains.

Ongoing Research Interests

Open questions I'm actively exploring

01

LLM Self-Repair via Compiler Feedback Loops

Extending adversarial repair beyond PDDL — applying compiler-guided correction to code generation, API calls, and structured data synthesis tasks where LLMs exhibit systematic failure modes.

02

Natural Language to Executable Test Suites

Translating plain-language specifications directly into runnable UI and API test cases, eliminating the need for human-authored test scripts and enabling non-technical stakeholders to define test coverage.

03

Context Management in Long-Horizon Agentic Tasks

Investigating strategies for maintaining coherent state across thousands of tokens: hierarchical summarization, memory gating, and selective retrieval for multi-session agentic workflows.

04

Unified Memory Architecture for Personalized AI

Designing layered memory systems that distinguish facts, preferences, and episodic context — with AI-driven memory suggestion and granular scope control across conversation and user levels.

05

Cross-Modal Automation: Vision + Language Models

Leveraging multimodal LLMs to interpret screenshots, UI state, and visual layouts for automation — removing the need for DOM access or accessibility trees in visual testing pipelines.

06

Prompt Stability Across LLM Version Upgrades

Analyzing behavioral drift in prompts as underlying models are updated — building evaluation harnesses that detect semantic regressions and auto-suggest prompt repairs for stable, version-resilient pipelines.
